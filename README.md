#  Bot Battlr App

Welcome to **Bot Battlr App**, that allows users to manage a collection of bots. Users can view available bots, enlist them into their army, release them, or permanently discharge them. The application interacts with a local JSON server to store and retrieve bot data.

##  Project Overview

Bot Battlr App is a web application that enables users to:

- Browse a collection of bots
- View individual bot details
- Enlist bots into their army
- Release or permanently discharge bots

All data is served from a local JSON server.
